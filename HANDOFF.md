## 2026-07-09 — Real models now drive the WF state machine

- **Diagnosis (Herman's catch):** with real models the workflow wasn't advancing and the harness "looked off" in dev mode. Root cause was the model under-filling structured output, three ways: (1) it invented field names (`theme_original`) because the contract promised an "injected course_state 模式" that was never actually injected — only the live snapshot was, and a fresh snapshot omits unset fields, so the model had no field dictionary → `bad_delta` strip → theme lost; (2) nothing told it to emit `completed_nodes`, so `completed_nodes` stayed `[]` → 工作流地图 never advanced and node-prereq/stage judgements had nothing to act on; (3) `wf_trace` (dev annotation) is model-emitted and the contract never asked for it. The harness WAS running (it stripped the bad field) — the model just fed it too little.
- **Fix A+B — contract** ([demo/src/prompts/contract.zh.md](demo/src/prompts/contract.zh.md)): added a `state_delta` writable-field dictionary (exact field names + shapes + enums, derived from `harness/schema/course-state.schema.json`) so the model writes `theme_resource.name`, not `theme_original`; and rule 8 requiring `state_delta.completed_nodes` (the WF node ids executed this turn). Folded into the existing contract file — NOT a new prompt section — so the 4-section assembly and `prompt-builder` byte-parity test still hold. Prompt grew ~5.0k→8.2k chars (cache absorbs most).
- **Fix C — server** ([demo/serve.mjs](demo/serve.mjs)): when the model emits no `wf_trace`, synthesize one from `state_delta.completed_nodes` + stage + `WF_NODES` names (dev-mode only); when the model declared no nodes, the trace says so honestly instead of showing nothing.
- **Verified live** (Zen `deepseek-v4-flash-free`, debug): model now writes `theme_resource {name:醒狮,…}`, `class_profile`, `teacher_resource_intent`, `pending_confirmations`, and `completed_nodes` (`["WF01","WF02","WF02b"]` one run) with **no bad_delta**; 工作流地图 advances (✓ WF01…) and the wf_trace annotation renders on real turns.
