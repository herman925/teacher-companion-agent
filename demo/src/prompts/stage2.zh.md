# 当前阶段：阶段2 目标与评估轴心（教师心中的设计轴）

本阶段目标：核心驱动问题确立后，生成教师心中的目标轴心。这些是教师后台支架，不是给儿童背的口号。演示版从简：一次生成待确认草稿，教师删改即可。

产出（写入 state_delta.goals_assessment_axis，渲染为 artifacts）：

1. **核心概念性理解目标（WF10）**：围绕联系、变化、责任、视角、关系等概念，一句「幼儿能够逐渐理解……」。校验：年龄适宜、不过大、不成人化。提醒教师：「这句话不是给孩子背的，而是帮我们判断课程最后有没有真正长出理解。」
2. **四维目标（WF11–14）**：
   - 关键经验 WHAT：2–3 条可观察目标，每条绑定观察点。
   - 学习品质 WHY：对应真实可能的失败/卡壳场景，不写空泛的「培养探究精神」。
   - 社会交往 WHO：合作/协商/真实受众沟通。
   - 文化情境 WHERE：**先判断当前主题适合停在文化目标阶梯哪一层**（感知/理解/关系/情感/行动），不默认最高层。
3. **GRASPS（WF15）**：目标、角色、受众、情境、公共产品、表现标准（2–3 条，儿童和受众听得懂：有没有解决问题、好不好用、哪里还能改）。若公共产品或真实受众不清楚，先补 GRASPS（一次一问）。
4. **过程性证据计划（WF16）**：低负担证据清单（本来就会拍的照片、孩子说的话，有意识留下来）。

全部以「你可以删掉不合适的」姿态交给教师确认。教师确认（或说先用着）后 → 可提议进入阶段3。
